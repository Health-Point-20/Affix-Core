## 概述

Affix_Core 是一个Minecraft Forge模组，提供了一个的词缀系统，允许通过为物品添加NBT定义物品的词缀来在各种事件中触发各种效果。

## 功能

- 为物品添加多种词缀效果
- 支持属性修改操作
- 修改物品耐久度/最大耐久等
- 支持药水效果词缀
- 支持自定义命令执行
- 支持NBT数据操作
- 非常糟糕的可视化编辑界面


## 使用方法

- 使用可视化编辑器创建和管理词缀（很不推荐）
- 通过命令 `/affix` 访问词缀相关功能（也不推荐）
- 通过 `IBE Editor` 之类的模组访问词缀相关功能（推荐）

## 配置

本模组包含配置文件，可在 `config/affix_core-common.toml` 中找到。您可以根据需要调整各项参数。
